## [3.0.0] - 2025-01-XX

### Добавлено

- **Новая дизайн-система Vailite**
  - Amber-Orange цветовая палитра (#f59e0b, #ea580c)
  - Тёмный фон #0c0a09
  - Glass-card эффекты с blur
  - Градиентные кнопки и текст

- **Анимации Framer Motion**
  - Плавающая иконка в header
  - Пульсация иконки анализа
  - Появление результатов с fade-in

- **Логотип Vailite**
  - Favicon в браузерной вкладке
  - Логотип в header страницы
  - Файл: `/public/vailite-logo.jpeg`

- **Эффект печатающегося текста**
  - Компонент `TypewriterText`
  - Мигающий курсор при печати
  - Плавное появление описания

### Изменено

- **globals.css** — полностью переработаны CSS-переменные
  - Primary: Amber (#f59e0b)
  - Accent: Orange (#ea580c)
  - Borders: полупрозрачные amber

- **page.tsx** — новый layout header
  - Логотип по центру
  - Иконка + заголовок в одну строку
  - Typewriter-эффект для описания

- **Компоненты** — обновлены под Vailite стиль
  - `ProfileInput`: amber-бордеры
  - `ResultDisplay`: glass-card, бейджи, анимации
  - `TelegramText`: скруглённые углы, amber-граница
  - `PdfDownload`: градиентная кнопка

### Техническое

- Добавлены CSS-классы: `.vailite-gradient`, `.vailite-gradient-text`, `.glass-card`, `.card-hover`
- Кастомный scrollbar в amber-тоне
- Backup версии 3: `/backup_v3_vailite_design/`

## [2.0.0] - 2025-01-XX

### Добавлено

- **Кликабельные кнопки в PDF**
  - Telegram: "Записаться на онлайн-звонок" → https://t.me/VadimCrypton
  - Сайт: "vailite.ru" → https://vailite.ru
  - Резервный контакт: @VadimCrypton в тексте

- **Динамический блок "Темы контента"**
  - Темы генерируются LLM под специализацию психолога
  - Формат: тема + заголовок + тип контента
  - Пример: "Тревожность" → "5 признаков, что тревога управляет вами" → экспертный ролик

- **Динамический кейс**
  - LLM генерирует кейс с похожей специализацией
  - Имя, цифры, результаты — релевантные психологу
  - Fallback на дефолтный кейс при ошибке

- **Новые типы в `types.ts`**
  - `ArticleTopic` — интерфейс для темы контента
  - `CaseStudy` — интерфейс для кейса

### Изменено

- Промпт `ANALYSIS_PROMPT` теперь требует генерацию `case_study`
- PDF-генератор использует динамические данные вместо захардкоженных
- API `/api/generate-pdf` передаёт `article_topics` и `case_study` в PDF

### Техническое

- Backup версии 2: `/backup_v2_dynamic/`
- Обновлён `package.json` до версии 0.2.0

## [1.0.0] - 2025-01-XX

### Добавлено

- **Базовая структура проекта**
  - Next.js 16 с App Router
  - React 19 + TypeScript
  - Tailwind CSS 4 + shadcn/ui

- **AI-анализ профиля**
  - Endpoint `/api/analyze` — анализ через LLM
  - Извлечение: имя, специализация, подход, опыт, боли
  - Интеграция с z-ai-web-dev-sdk

- **Генерация PDF**
  - Endpoint `/api/generate-pdf` — создание PDF
  - Python + ReportLab для генерации
  - Поддержка кириллицы через DejaVu Sans
  - Тёмная тема с золотыми акцентами

- **Генерация текста для мессенджеров**
  - Короткое приветствие для Telegram/WhatsApp
  - Персонализация под профиль

- **Frontend компоненты**
  - `ProfileInput` — ввод JSON профиля
  - `ResultDisplay` — отображение результата
  - `TelegramText` — копируемый текст для мессенджеров
  - `PdfDownload` — кнопка скачивания PDF

- **Структура PDF (4 страницы)**
  - Страница 1: Обращение, профиль, причины
  - Страница 2: Форматы, автоматизация, темы, кейс
  - Страница 3: Предложение, что входит, сравнение
  - Страница 4: CTA, контакты, P.S.

### Техническое

- Python virtual environment в `/venv/`
- Зависимости: reportlab, pillow
- Backup версии 1: `/backup_v1/`
